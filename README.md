# DApp - Willow

## Description
Decentralized Real Estate Application. Smart contracts written in Solidity. Ethers v5 used for EVM communication. Frontend built with React.js. Testing, compilation, and deployment done with Hardhat @2.12.0.

* #### Build in progress.
